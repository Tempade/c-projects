This project contains 6 different files:
	1.) redact .cpp/.h		[ Contain functions to replace specific chars with "#" ]
	2.) self_destructing .cpp/.h	[ Contains functions to create a message with a specific number of views ]
	3.) test_redact.cpp		[ Allows user to test the redact files ]
	4.) test_self_destructing.cpp	[ Allows user to test the self_destructing files ]

This project was originally created for CSE 232 at MSU
