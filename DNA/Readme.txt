################################################
#   		DNA Information		       #
################################################

This program will provide the user with useful information about a given DNA sequence.
When a DNA sequence is inputted, the user will recieve the following back:
	1.) The reverse complement DNA sequence
	2.) The RNA transcript for the original sequence
	3.) The codon reading frames
	4.) The amino acid sequence for each of the above
	5.) The longest open reading frame (if available)

A valid DNA sequence only uses the following letters: 'A', 'T', 'C', and 'G'

*Program was originally created for the class CSE 232 at MSU
